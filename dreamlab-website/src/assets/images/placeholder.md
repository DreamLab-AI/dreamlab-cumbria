## Image Specifications

### Optimization Guidelines
- **Format**: Use WebP with JPG fallback for compatibility
- **Compression**: Optimize for web (60-80% quality)
- **Responsive**: Provide multiple sizes for different screen densities
- **Alt Text**: All images require descriptive alt text for accessibility

### Recommended Sizes
- **Hero Images**: 1920x1080px minimum
- **Gallery Images**: 800x600px minimum
- **Card Images**: 400x300px
- **Profile Photos**: 300x300px (square)
- **Icons**: 24x24px, 32x32px, 48x48px (SVG preferred)

### File Naming Convention
Use descriptive, lowercase names with hyphens:
- `accommodation-main.jpg`
- `training-led-screen.jpg`
- `team-ceo-profile.jpg`
